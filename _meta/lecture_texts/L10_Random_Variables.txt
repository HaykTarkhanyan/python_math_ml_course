probabilities.

вҖқWhat is the probability that X = k?вҖқ
i.e.
P[X = k]
is to denote it as:
pX(k)
and call it the PMF of X.
Definition
The function which takes k and returns the probability that X = k:
pX(k) = P[X = k]
is called the probability mass function or PMF of X.
If you know the PMF of X, you know everything about the distribution of
X (i.e. how likely each value is).
