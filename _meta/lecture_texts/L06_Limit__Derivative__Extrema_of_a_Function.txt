Extrema of a Function
Theorem 1 (f ′′ at one point)
If f ′(x0) = 0 and there exists finite f ′′(x0), then
1 If f ′′(x0) > 0, then x0 is a local minimum point,
2 If f ′′(x0) < 0, then x0 is a local maximum point.
Theorem 2 (f ′ at multiple points)
If for some δ > 0, f is differentiable in the intervals (x0 −δ, x0) and
(x0, x0 + δ) and continuous at x0, then
1 If f ′(x) > 0 for x ∈(x0 −δ, x0) and f ′(x) < 0 for x ∈(x0, x0 + δ),
then x0 is a local maximum point.
2 If f ′(x) < 0 for x ∈(x0 −δ, x0) and f ′(x) > 0 for x ∈(x0, x0 + δ),
then x0 is a local minimum point.
3 If f ′(x) doesn’t change its sign, then x0 is not an extremum point.
