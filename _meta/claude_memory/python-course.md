## Python Course (python/)

18 Jupyter notebooks covering Python fundamentals through advanced OOP:

**Phase 1 — Fundamentals (01-06):** print/variables/numbers → conditionals → strings/lists/range → loops → methods/comprehensions → tuples/sets/dicts

**Phase 2 — Functions & System (07-10):** function basics (def, scope, args) → advanced functions (lambda, map/filter, *kwargs, LEGB) → terminal/packages/file I/O → git/conda/PEP8

**Phase 3 — Error Handling & OOP (11-17):** exception handling (try/except, EAFP) → streamlit/recursion → decorators → classes → inheritance/polymorphism → encapsulation/abstraction → dataclasses/iterators/generators/context managers

**Phase 4 — Capstone (18):** YouTube Transcript Translator — full OOP app using pytubefix, googletrans, deepl, async/await, ABC

## Notebook Structure Pattern (consistent across all)
- YAML header (title, lightbox, code-fold)
- Armenian cultural photograph with Unsplash attribution
- Google Colab badge link
- Song/quote reference
- 📺 Video sections: Theory (2025 + 2023 recordings), Practical, Solution
- 📚 Material: detailed content with code cells
- 🛠️ Practical problems
- 🏡 Homework: Profound.academy links
- 🎲 Cultural references (music, videos — Armenian + international)
- Flag counter analytics

## Python Libraries Course (python_libs/)
18 modules covering the data science stack (separate from main Python course).

## Mini Projects (python/mini_projects/)
4 student projects: approximate_pi (Monte Carlo), telegram_data (JSON), text_corpus_classes (OOP), youtube_playlist (analytics)

- Python 3.10+ (uses match/case, type hints, dataclasses)
