\section{The recurrence idea}

\begin{frame}[plain]
  \vfill
  \begin{center}
    {\Large\color{popblue}\textbf{The recurrence idea}}\\[0.9em]
    \begin{minipage}{0.82\textwidth}\centering\small
      Read like a human: one word at a time, with memory.
    \end{minipage}
  \end{center}
  \vfill
\end{frame}

\begin{frame}{One new arrow}
  \begin{columns}[T]
    \column{0.52\textwidth}
    How a human reads: word by word, retaining a running summary - the \textbf{state}.
    \vskip 6pt
    From L14's hidden layer $\mathbf{z}=\sigma(\mathbf{W}^\top\mathbf{x}+\mathbf{b})$ to
    \[
      \mathbf{z}^{[t]} = \sigma\big(\textcolor{armred}{\mathbf{V}^\top \mathbf{z}^{[t-1]}}
        + \mathbf{W}^\top \mathbf{x}^{[t]} + \mathbf{b}\big):
    \]
    a single new term, the \textbf{hidden-to-hidden weights} $\mathbf{V}$.
    \vskip 4pt
    \fcolorbox{popblue}{popblue!6}{\parbox{0.95\linewidth}{\small
      $\mathbf{z}^{[t]}$ is a function of the current input \emph{and} the previous
      state - by recurrence, a summary of \textbf{everything read so far}.}}
    \column{0.48\textwidth}
    \begin{center}
      \begin{tikzpicture}[font=\scriptsize, >=Latex, node distance=0.5cm,
          n/.style={draw, circle, minimum size=0.7cm}]
        \node[n, fill=armblue!12] (x) at (0,0) {$\mathbf{x}^{[t]}$};
        \node[n, fill=paramgreen!15] (z) at (2.0,0) {$\mathbf{z}^{[t]}$};
        \node[n, fill=armorange!16] (f) at (4.0,0) {$f^{[t]}$};
        \node[n, fill=paramgreen!10, xshift=-0.3cm] (zprev) at (2.0,-1.8) {$\mathbf{z}^{[t-1]}$};
        \draw[->] (x) -- (z) node[midway, above] {$\mathbf{W}$};
        \draw[->] (z) -- (f) node[midway, above] {$\mathbf{U}$};
        \draw[->, armred, very thick] (zprev) -- (z)
          node[midway, right, xshift=1pt] {\textcolor{armred}{$\mathbf{V}$}};
      \end{tikzpicture}
      \vskip 2pt
      {\footnotesize \textcolor{armred}{Red}: the one new arrow, hidden to hidden.}
    \end{center}
  \end{columns}
\end{frame}

\begin{frame}{Picking the activation: why tanh}
  \begin{columns}[T]
    \column{0.48\textwidth}
    \footnotesize Three candidates for the state's activation - only one is the RNN
    default:
    \begin{itemize}\footnotesize
      \item \textbf{tanh} (default): bounded in $(-1,1)$ - the state literally
        \textbf{cannot blow up} across steps; zero-centered, so both signs flow.
      \item \textbf{sigmoid}: also bounded, but $|\sigma'|\le 1/4$ everywhere - one
        more shrinking factor on top of $\mathbf{V}$ every step (the "full derivation,
        part 2" frame ahead uses exactly this bound).
      \item \textbf{ReLU}: unbounded above - feed that back through $\mathbf{V}$ every
        step and the state can genuinely \textbf{explode}, not just saturate.
      \end{itemize}
    \vskip 2pt
    \fcolorbox{armred}{armred!8}{\parbox{0.95\linewidth}{\footnotesize
      \textbf{Dead ReLU}, one line: a unit whose pre-activation is always $\le 0$
      outputs zero forever, and its gradient too - it never updates again.}}
    \column{0.52\textwidth}
    \begin{center}
      \includegraphics[width=0.92\linewidth]{activation_zoo.pdf}
    \end{center}
  \end{columns}
\end{frame}

  \begin{columns}[T]
    \column{0.44\textwidth}
    \begin{center}\includegraphics[width=0.95\linewidth]{bidir_rnn.png}\end{center}
    \vskip 1pt
    {\scriptsize The \textbf{forward} states $z^{[t]}$ run left$\to$right; the
    \textbf{backward} states $g^{[t]}$ run right$\to$left; each output $f^{[t]}$ reads
    \emph{both}.}
    \column{0.56\textwidth}
    A plain RNN only sees the \textbf{past}. But the right reading of a word often depends on
    what comes \emph{after}:
    \vskip 3pt
    \begin{itemize}\small
      \item \emph{``\textbf{Teddy} Roosevelt''} vs \emph{``\textbf{Teddy} bears''} - is

    \end{itemize}
    \vskip 4pt
    A \textbf{bidirectional} RNN runs one RNN each way and concatenates their states, so
    \textbf{every position sees the whole sequence}.
    \vskip 6pt
    \fcolorbox{armblue}{armblue!8}{\parbox{0.95\linewidth}{\small
      Great for \textbf{encoding} a sequence you already have (tagging, sentiment, the
      encoder half of translation). The catch: it needs the \emph{whole input up front}, so
      it cannot \textbf{generate} or stream token by token - the setting L21 cares about.}}
  \end{columns}

\begin{frame}{Predict-first: the fading number}
  \begin{columns}[T]
    \column{0.42\textwidth}
    Strip the net down to $1$-d to see the decay cleanly: a scalar state
    $h^{[t]}=V h^{[t-1]}+W x^{[t]}$, no vectors, no activation. On a 30-word review,
    how much does token $1$ influence the state at token $30$?
    \[
      \frac{d h_{30}}{d h_1} = V^{29} = 0.8^{29}
    \]
    \textbf{Guess the value.}
    \pause
    \vskip 4pt
    \fcolorbox{armred}{armred!8}{\parbox{0.95\linewidth}{\small
      $0.8^{29}\approx \mathbf{0.0015}$. And if $V=1.25$? $1.25^{29}\approx\mathbf{646}$.
      One multiplication, repeated, either starves or explodes.}}
    \column{0.58\textwidth}
    \begin{center}
      \includegraphics[width=\linewidth]{vanish_curve.pdf}
    \end{center}
  \end{columns}

\begin{frame}{The full derivation, part 1}
  For vectors, the state update is
  $\mathbf{z}^{[t]} = \sigma(\mathbf{V}^\top\mathbf{z}^{[t-1]}+\mathbf{W}^\top\mathbf{x}^{[t]}+\mathbf{b})$,
  so by the chain rule:
  \[
    \frac{d\mathbf{z}^{[t]}}{d\mathbf{z}^{[t-1]}}
      = \text{diag}\big(\sigma'(\cdot)\big)\,\mathbf{V}^\top
      = \mathbf{D}^{[t-1]}\,\mathbf{V}^\top
  \]
  Chain this back to step $1$:
  \[
    \frac{dL}{d\mathbf{z}^{[1]}} = \frac{dL}{d\mathbf{z}^{[t]}}\;
      \mathbf{D}^{[t-1]}\mathbf{D}^{[t-2]}\cdots\mathbf{D}^{[1]}\,(\mathbf{V}^\top)^{t-1}
  \]
  \vskip 8pt
  \fcolorbox{popblue}{popblue!6}{\parbox{0.97\linewidth}{\small
      Every path back to an early time-step runs through powers of the \textbf{same}
      matrix $\mathbf{V}^\top$. For any $i<t$: the term $(\mathbf{V}^\top)^{t-i}$ appears.}}
\end{frame}

\begin{frame}{The full derivation, part 2 - eigenvalues}
  The \textbf{largest eigenvalue} $|\lambda_{\max}|$ of $\mathbf{V}$ decides the fate of
  $(\mathbf{V}^\top)^{t-i}$:
  \begin{itemize}
    \item $|\lambda_{\max}| < 1 \Rightarrow$ \textbf{vanishing}.
    \item $|\lambda_{\max}| > 1 \Rightarrow$ \textbf{exploding}.
  \end{itemize}
  \vskip 6pt
  \fcolorbox{armred}{armred!8}{\parbox{0.97\linewidth}{\small
      \textbf{Worse than a deep MLP.} An MLP has a \emph{different} matrix per layer -
      errors can partially cancel. An RNN multiplies by the \textbf{same} $\mathbf{V}$
      at every step, so the effect compounds cleanly. The sigmoid's $\mathbf{D}$ factors
      only make it worse ($|\sigma'|\le 1/4$).}}
  \vskip 6pt
  {\footnotesize Callback: L15 already met vanishing gradients in deep MLPs. Same
  disease, worse here - it is the \emph{same} matrix every step.}
\end{frame}

\begin{frame}{Exploding is the easy half}
  \begin{columns}[T]
    \column{0.5\textwidth}
    L15 already gave us the fix (recall "Saddles and cliffs"): cap the gradient's norm
    at a threshold $h$:
    \[
      \text{if } \|\nabla\| > h:\quad \nabla \leftarrow \frac{h}{\|\nabla\|}\,\nabla
    \]
    \vskip 4pt
    \fcolorbox{armorange}{armorange!12}{\parbox{0.95\linewidth}{\small
      One formula, solved problem. Exploding gradients are the \emph{easy} half of this
      story.}}
    \column{0.5\textwidth}
    \begin{center}
      \begin{tikzpicture}[font=\scriptsize, >=Latex]
        \draw[dashed, gray] (0,0) circle (1.3);
        \node[gray] at (0,-1.6) {norm threshold $h$};

    \end{center}
  \end{columns}
\end{frame}

\section*{Recap}

\begin{frame}{Recap}
  \begin{itemize}
    \item Text refuses to fit a fixed input: length varies, order matters - every
      shoehorn hack loses one or the other.
    \item State $=$ a running summary; \textbf{one new weight matrix} $\mathbf{V}$ turns
      a dense layer into an RNN.
    \item Unrolled $=$ a deep network with \textbf{tied weights} - autograd handles it
      unchanged.
    \item Same-matrix powers $\Rightarrow$ gradients \textbf{vanish} (derived, not
      asserted) or explode.
    \item Clipping fixes exploding. \textbf{Nothing yet fixes vanishing.}
  \end{itemize}
  \vskip 6pt
  \fcolorbox{paramgreen}{paramgreen!8}{\parbox{0.97\linewidth}{
      \textbf{Next:} 1997's answer - a memory built from \textbf{addition}, not
      multiplication: the LSTM. It works. And it still will not be enough (L21).}}
\end{frame}
