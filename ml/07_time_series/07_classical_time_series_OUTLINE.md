# 07 Classical Time Series — outline

Lecture 1 of the time-series chapter. 22 frames. Clean PDF only (not delivered
yet, so no `_notes.pdf`). Bridges from all prior i.i.d. supervised lectures.

## Sections / frames

1. **Cold open** — "every model so far assumed rows were interchangeable"; time
   series breaks it. Two-lecture roadmap.
2. **What makes time series different** — vocabulary (horizon, uni/multivariate);
   the four components (trend/season/cycle/noise); STL decomposition. *(figs:
   ts_anatomy, stl_decomposition)*
3. **Stationarity and differencing** — definition; stationary vs 3 non-stationary
   panels; differencing + ADF test. *(figs: stationarity, differencing)*
4. **Autocorrelation** — ACF vs PACF; reading AR/MA orders. *(fig: acf_pacf)*
5. **ARIMA family** — AR, MA, ARMA, ARIMA(p,d,q), SARIMA(p,d,q)(P,D,Q)_m; forecast
   with prediction interval; auto_arima flagged. *(fig: arima_forecast)*
6. **Exponential smoothing** — SES → Holt → Holt-Winters. *(fig: exp_smoothing)*
7. **Baselines and evaluation** — naive / seasonal-naive / drift; MAE/RMSE/MAPE/MASE;
   time-ordered holdout + backtesting; when classical vs ML wins (teaser).
8. **Recap** + green "Next: ML for time series" box.
