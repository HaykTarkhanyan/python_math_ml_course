  \vfill
  \begin{itemize}\small
    \item \textbf{The flip.} True convolution \emph{flips} the kernel before sliding. The 1D
      animation (next) shows the flip-and-slide that L16's footnote only named.
    \item \textbf{The convolution theorem, proved.} L16 stated
      $\mathcal{F}\{f \ast g\} = \mathcal{F}\{f\}\,\mathcal{F}\{g\}$; here is the derivation

      side-by-side with convolution -- and why \texttt{nn.Conv2d} computes cross-correlation

  \end{itemize}

% OUR BRIDGE FRAME: convolution -> cross-correlation (makes L16's footnote precise)
\begin{frame}{From convolution to cross-correlation}
  You have just seen \emph{true} convolution: flip the kernel, then slide. What every
  deep-learning framework actually computes drops the flip -- that is
  \textbf{cross-correlation}.
  \vfill
  \begin{itemize}\small
    \item Same slide-and-sum, \emph{without} reversing the kernel.
    \item This is L16's footnote made precise: \texttt{nn.Conv2d} is cross-correlation
      wearing convolution's name.
    \item It changes nothing in practice -- the kernel is \textbf{learned}, so the network
      would just learn the flipped weights either way.
  \end{itemize}
  \vfill
  \fcolorbox{armorange}{armorange!12}{\parbox{0.95\linewidth}{\small
    The next stretch is LMU's formal cross-correlation: definition, its own click-through
    animation, and the final side-by-side comparison.}}
\end{frame}

% OUR CLOSING FRAME: back to the main thread
\begin{frame}{Back to L16}
  \begin{itemize}\small
    \item \textbf{Convolution} is a flip-and-slide dot product; it is commutative and has a
      clean identity in frequency (the convolution theorem).
    \item \textbf{CNNs compute cross-correlation} -- the same operation without the flip --
      because the kernel is learned, so the flip is irrelevant.
    \item The frequency view is why a kernel is a \textbf{frequency filter}: box blur $=$
      low-pass, Sobel $=$ high-pass (L16, section 2).
  \end{itemize}
  \vfill
  \fcolorbox{paramgreen}{paramgreen!8}{\parbox{0.97\linewidth}{\small
    None of this changes how you build or train a CNN. It is the mathematics that justifies the
    one line L16 asked you to take on faith. Back to the main thread: \textbf{L17, the
    architectures.}}}
