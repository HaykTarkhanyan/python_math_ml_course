\begin{frame}{A cleaner residual: the output gradient is $f - y$}
  LMU used the \textbf{L2 loss} on class labels to keep the algebra simple. In practice,
  classification uses \textbf{cross-entropy} -- and then the output-layer gradient collapses to
  something you already know:
  \vfill
  \[
    \text{sigmoid output } + \text{ cross-entropy} \;\;\Longrightarrow\;\;
    \frac{\partial L}{\partial f_{\text{in}}} = f - y
  \]
  \vfill
  \fcolorbox{armblue}{armblue!8}{\parbox{0.97\linewidth}{\small
    \textbf{The same ``residual'' from earlier in the course.} Linear regression, logistic
    regression, and now a neural net: the output signal is \textbf{prediction $-$ target}.
    Everything to its left is that residual, passed backward and scaled by local derivatives.}}
\end{frame}

% --- OUR SPRINKLE (after the recursion): its cost -- vanishing gradients ---
\begin{frame}{What this recursion costs: vanishing gradients}
  LMU's recursion $\;\delta^{(i)} = \mathbf{W}^{(i+1)}\delta^{(i+1)} \odot
  \sigma'(\mathbf{z}^{(i)}_{\text{in}})\;$ multiplies by $\sigma'$ at \emph{every} layer on the
  way back -- and that factor is small:
  \vfill
  \begin{columns}[T]
    \column{0.52\textwidth}
    \begin{center}\includegraphics[width=\linewidth]{fig/activation_grads.pdf}\end{center}
    \column{0.48\textwidth}
    \begin{itemize}\small
      \item Sigmoid: $\sigma'\le 0.25$. After $k$ layers the gradient is scaled by $\le
        0.25^{k}$ -- it \textbf{vanishes exponentially} with depth.
      \item $0.25^{10}\approx 10^{-6}$: a $10$-layer sigmoid net shrinks the gradient a
        \emph{millionfold} before it reaches layer $1$.
      \item \textbf{ReLU} ($\sigma'=1$ for $z>0$) largely fixes this -- the default hidden
        activation.
    \end{itemize}
  \end{columns}
  \vfill
  \fcolorbox{armred}{armred!8}{\parbox{0.97\linewidth}{\small
    The \textbf{vanishing-gradient} problem is a direct consequence of the backprop recursion
    LMU just derived -- and a big reason deep nets were hard to train before ReLU.}}
\end{frame}

\begin{frame}{Recap: how a network learns}
  \begin{itemize}\small
    \item \textbf{Objective:} minimise the empirical risk by \textbf{gradient descent};
      \textbf{SGD} does it on mini-batches -- cheaper, noisier, and often generalises better.
    \item \textbf{Backprop} $=$ the chain rule on the computational graph, run \emph{backward
      once}. Each layer's error signal $\delta$ is computed from the layer above and reused:
      ``$\delta\times$ local derivative'', ``$\delta\times$ input''.
    \item That reuse is why \emph{one} backward pass gives \textbf{all} gradients -- the naive
      alternative is one forward pass per weight.
    \item \textbf{You won't code it by hand:} frameworks (PyTorch, TensorFlow) build the graph
      and run backprop when you call \texttt{loss.backward()} -- on \textbf{GPUs/TPUs} that do
      the matrix math massively in parallel.
  \end{itemize}
  \vfill
  \fcolorbox{paramgreen}{paramgreen!8}{\parbox{0.97\linewidth}{\small
    We can now \emph{train} a network. What remains: keeping it from \textbf{overfitting}
    (regularization), and \textbf{optimizing} faster than plain SGD -- the next two chunks.}}
\end{frame}
