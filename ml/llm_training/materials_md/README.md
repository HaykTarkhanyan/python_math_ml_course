# materials_md - readable text versions

Plain-text / markdown versions of everything in [`../materials/`](../materials/), for fast
reading, search, and grep (the PDFs and SPA-HTML originals are awkward to read directly).
Same `01`-`10` numbering as the originals.

Generated **2026-06-23**:

- `papers/*.txt` - extracted from the PDFs with `pdftotext` (poppler). Plain text in reading order.
- `blogs/*.md` - converted from the HTML snapshots with `pandoc -f html -t gfm`.

## Where to read figures (each file starts with a map)

Because figures/plots don't survive text extraction, every file begins with a
**"where to read" map** so you know where to look in the original:

- **Papers (`.txt`):** a **Figure & table map** listing each figure/table, its caption, and
  its **PDF page number**. Open that page in the original with the Read tool, e.g.
  `Read("../materials/papers/03_llama3_2407.21783.pdf", pages="8")`.
- **Blogs (`.md`):** an **image / figure map** listing each image by the section it appears in,
  with a **direct URL** to the image on the HuggingFace host (open it in a browser / fetch it).

The maps are auto-generated by pattern matching, so they catch most but not every figure, and
long captions are truncated. They are pointers, not a substitute for the original.

## Fidelity caveats

- **Text only.** Figures, plots, equations rendered as images, and tables can be lost,
  garbled, or flattened. For anything precise (numbers in a table, a figure, an exact
  formula), check the original in [`../materials/`](../materials/).
- **Papers (`.txt`):** math notation and multi-column tables may be imperfect.
  `10_qwen3` contains some Chinese glyphs that `pdftotext` could not map (harmless warning;
  English text is intact).
- **Blogs (`.md`):** the playbooks' interactive plots/iframes don't convert - only their
  surrounding prose and headings are captured. A few asset-fetch warnings during conversion
  were expected and harmless.

These are a convenience layer. The originals in `../materials/` remain the source of truth.
